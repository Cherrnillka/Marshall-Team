Задание 2: Основы CSS

Изучим основы CSS для стилизации страницы и создания простых эффектов.

Что вам нужно сделать:

  • Создайте страницы с базовой стилизацией.
  • Используйте CSS для стилизации различных элементов.
  • Экспериментируйте с различными свойствами CSS.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Моя страница</title>
<style>
    body { background-color: #f0f0f0; font-family: Arial, sans-serif; }
    h1 { color: #e65100; }
</style>
</head>
<body>
    <h1>Добро пожаловать!</h1>
    <p>Это пример страницы.</p>
</body>
</html>

Что дальше?

Попробуйте изменить цвета элементов, добавьте фоновые изображения, создайте анимации. Экспериментируйте с CSS-свойствами, чтобы улучшить визуальный вид страницы.
