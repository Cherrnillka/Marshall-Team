Задание 5: Ваша тема

Описание задания с примером использования нового стиля и цветов.

Что вам нужно сделать:

  • Используйте новые цвета для стилизации элементов.
  • Попробуйте экспериментировать с фонами и анимациями.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Моя страница</title>
<style>
    body { background-color: #fce4ec; font-family: Arial, sans-serif; }
    h1 { color: #d81b60; }
</style>
</head>
<body>
    <h1>Добро пожаловать!</h1>
    <p>Это пример страницы.</p>
</body>
</html>

Что дальше?

Попробуйте создать свои собственные стили и добавьте эффекты для улучшения визуальной привлекательности страницы.
