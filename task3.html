Задание 3: Основы JavaScript

В этом задании мы познакомимся с основами JavaScript. Мы научимся создавать переменные, работать с функциями и событиями.

Что вам нужно сделать:

  • Создать переменную и вывести её значение в консоль.
  • Научиться использовать простые функции для выполнения действий.
  • Создать кнопку, которая при нажатии выводит сообщение.

Пример кода:


// Создаем переменную
let greeting = "Привет, мир!";

// Выводим в консоль
console.log(greeting);

// Функция для вывода сообщения
function showMessage() {
    alert("Привет, это ваше первое событие в JavaScript!");
}
